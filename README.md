## DND Manager

### Requirements
- Python 3.10
- Docker >=  24.0.5
- docker-compose >= 2.23.3

### Install
```shell
python -m pip install virtualenv
make install-venv
make install
```

### Run
Start containers  
```shell
make up
```

### Test
```shell
make test
```

### Coverage
```shell
make coverage
```
